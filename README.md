## ゲームの内容
（ここにゲームの内容説明を記入する）
プレイヤーは〇・△・□の3種類の形に変身しながら、3つのレーンを左右に移動して上から降ってくるノーツをキャッチするゲーム。
降ってくるノーツと同じレーン・同じ形でキャッチするとスコアとコンボが加算され、コンボ数が増えるほど高得点を狙える。
制限時間終了時にはスコアが結果画面に表示され、難易度ごとのランキングに記録される。
異なる形で取る、またはミスを重ねるとミス数が増加する。ミス回数が上限に達するとゲームオーバーとなる。
制限時間内にできるだけ高いスコアを目指すゲーム。

## 操作説明
- （例）矢印キー、若しくはAとDキーで左右に砲台を移動する
- （例）スペースキーを押すとビームを敵に向けて発射する
- ←キー(左矢印キー)と→キー(右矢印キー)でプレイヤーを左右3レーンに移動する
- ↓キー(下矢印キー)で〇→△→□の順番に変身する
- ↑キー(上矢印キー)で□→△→〇の逆順に変身する
